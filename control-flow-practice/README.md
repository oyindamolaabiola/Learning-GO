### Loop Patterns

1. Counter loop
    - internal or external
    - increasing or decreasing
    - count up / down
    - first N items
2. Sentinel loop
    - stop on value or condition
    - stop on 0 / negative / "stop‚Äù
3. Accumulator
    - sum / average
    - count
    - max / min (running comparison)
4. Filter
    - skip / select values (continue / if)
5. Step loop
    - non-unit increments
    - i += 2, i += 5
6. Shifting (Top-N)
    - preserve top K values
    - move values to keep top 3
7. Nested loops
    - rows + columns
    - multi-dimensional repetition
8. Map-based uniqueness
    - track seen values
